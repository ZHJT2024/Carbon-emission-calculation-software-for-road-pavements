# Road Pavement Carbon Emission Calculation Software

This repository documents the **Road Pavement Carbon Emission Calculation Software**, a life-cycle assessment (LCA) platform developed by the authors to quantify the climate, health, and economic impacts of road pavement systems.

To protect intellectual property and ensure responsible use, the **source code is not publicly released**. Instead, the software is provided as an online platform for calculation, verification, and policy-oriented assessment.

---

## Software overview

The Road Pavement Carbon Emission Calculation Software is designed to support **life-cycle benefit analysis of road pavements**.
